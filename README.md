# Prismagram

Instagram clone with Express, Prisma, React, and React Native

## User Stories

- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [x] Create account
- [x] Log in - Request secret
- [ ] Log in - Confirm secret
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
