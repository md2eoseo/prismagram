# Prismagram

Instagram clone with Express, Prisma, React, and React Native

## User Stories

- [x] Create account
- [x] Log in - Request secret
- [x] Log in - Confirm secret
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location, caption
- [x] Follow User
- [x] Unfollow User
- [x] See user profile
- [x] See MY profile
- [x] See the full post
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
